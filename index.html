Move

Commands (prototype)

  • look — reprints the current room description
  • go <direction> — move (north, south, east, west, north east ...)
  • take <item> — pick up an item
  • leave <item> — drop an item
  • inventory — show inventory
  • use <item> — attempt to use an item
  • give <item> to <npc> — give an item
  • talk to <npc> — talk to caretaker / barista / scientist / puppy
  • build <thing> — build in the workshop (cart, birdhouse, shelf)
Prototype — gameplay logic will be in engine.js.
